AIPU hardware roadmap

Performance is set by memory bandwidth, which is set by the silicon's IO/PHY budget, which is set by the rung and its funding. Only rung ① is near-term buildable; rungs ② and ③ are funding-gated projections. Every tok/s figure is an estimate [EST].

The RTL is the same across all three rungs — only the bandwidth changes. The FPGA fit and routed Fmax are measured (see Board); there is no running board yet, so every tok/s is modeled until bring-up.

02

Rung detail

The near-term demo is rung ①; rungs ② and ③ are projections gated on funding and, for ②, on a correctness contingency.

① Prove-it (now)

Low-end FPGA (Kintex US+ KU3P class) with a DDR4 hot-set cache. Storage scales from 1–2 NVMe (~7–14 GB/s) up to ~14 striped drives (~100 GB/s).

~0.5–1 … ~5–8 tok/s [EST]
correctness bit-exact throughout cost ~$1–2k box funding self / minimal status the near-term demo

The FPGA fit is measured — see Board.

② Custom board (post-raise)

Mid-range FPGA (Versal / Agilex / HBM-class US+) on a custom board with DDR5 8–12 channels or HBM (~400 GB/s – 1 TB/s).

~15–40 tok/s [EST]
correctness contingent cost ~$3–6k funding seed

Contingent: needs an expert-cache hit rate OR non-bit-exact pruning.

③ SoC / ASIC (at scale)

Custom silicon. Primary design point: 512 GB LPDDR5X full residency (16×32 GB, 1024-bit, ~1.1 TB/s). The whole ~467 GB checkpoint is resident; cold store is one M.2 NVMe (~70 s boot).

≈80 tok/s [measured-inputs EST]
upside ~95 if MTP hits published depth ceiling ~120 HBM (aspirational) correctness bit-exact (residency ⇒ h=1) cost ~$1.8–2.4k funding Series B+ / volume

04

Open items

The gaps between here and rung ③, taken from the repo. Most sit at the physical / process boundary an RTL repo cannot close.

Whole-runtime GGUF cross-check

The ~467 GB GLM file run end-to-end and compared against llama.cpp's runtime argmax. Needs a GPU / large-memory host.

Board bring-up

The FPGA fit is measured in-repo; running on a physical board needs the board + a pin XDC + a MIG bridge.

Vendor PHY IP integration

DDR5 / NVMe (PCIe) / USB-C PHYs are TB-stubbed. Licensed PHY IP is integrated and signed off at the silicon rung.

ASIC tapeout

Custom-silicon tapeout is the rung-③ volume endgame.
