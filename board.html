AIPU — 개인용 프런티어 LLM 어플라이언스

753B급 오픈 모델(GLM-5.2, 4-bit 양자화 ~467GB)을 통째로 메모리에 올려 오프라인으로 구동하는 단일 사용자 박스 — 개념 설계

메모리 LPDDR5X 512GB · 1024-bit · ~1.1TB/s
저장 NVMe 1TB · 부팅 시 모델 로드 ~70초
속도 ~70–95 tok/s [추정] — 상용 API급 체감
전력 ~40–60W · 저소음
연결 USB-C 하나로 PC/폰과

① 메인보드 배치 (120 × 80 mm)

고속 광폭 메모리 버스(1024-bit)를 SoC 패키지 내부에서 완결하는 구조 — 보드에는 전원과 저속 I/O만 나와, 신용카드 두 장 크기로 작아집니다.

본 문서는 개념 설계입니다. 성능·비용 수치는 대역폭 루프라인 추정치이며(U(K)·수락률 입력은 GLM-4.5-Air 실측), 프로세서 RTL은 시뮬레이션에서 전체 포워드가 골든 참조와 비트 단위 동일(1,155 테스트), 디퀀트 계층은 실제 공개 GGUF에서 llama.cpp 자체 디퀀트 코드와 비트 단위 동일(1.046억 가중치)함이 검증되었습니다. FPGA는 배치·배선(fit)까지 실측 — 고정 커밋에서 측정하고 이후 변경은 형식 등가 증명으로 유지 — 실보드 구동 데모는 다음 단계입니다.
